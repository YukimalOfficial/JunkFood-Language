JunkFood Language SDK 使い方説明書
===================================

■ 概要
C言語をベースにした、独自のジャンクフード構文を持つ本格派コンパイラ環境です。

■ コンパイル方法
Windows ターミナル等を開き、以下のコマンドを実行します。
  python pcc.py [ファイル名.jf]

■ 全文法リファレンス
・order("文字列", ...);          : 画面へのテキスト出力（printf互換）
・set 変数名 = 値;               : 整数型（int）の変数宣言と代入
・pancakes 変数名 = 小数;         : 小数型（float）の変数宣言と代入
・potatochips&coke(回数) { ... } : 指定回数の高速ループ処理（カウンタ変数「bite」が自動使用可能）
・hotdog (条件) { ... }          : 条件式が真の間ループする処理（while互換）
・hamburger (条件) { ... }       : 条件分岐（if）
・friedpotatoes { ... }          : 条件分岐（else）
・popcorn (変数) { ... }         : 多方向の条件分岐（switch互換）
    - flavor 値:                 : 条件の分岐先（case）
    - finish;                    : 分岐からの脱出（break）
    - dustbox:                   : すべての条件に合致しない場合（default）
・pizza[要素数] 変数名;           : 静的配列の宣言（int型配列）
・icecream(ミリ秒);              : 指定した時間だけ処理を一時停止（Sleep互換）
・nugget 関数名() { ... }        : 戻り値なしの関数定義（void）
・nugget(型) 関数名() { ... }    : 戻り値ありの関数定義
・donut 値;                      : 関数から値を返却する（return互換）
